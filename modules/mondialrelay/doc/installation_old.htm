Installation du module Mondial Relay

Si vous disposez d’une des versions Prestashop

1.2.0.x / 1.2.1.x / 1.2.2.x / 1.2.3.x/ 1.2.4.x / 1.2.5.x  cliquez ici  

 

Si vous disposez d'une autre version

Copier les fichiers modules dans le dossier modules puis copier le fichier tabs/AdminMondialRelay.php  dans le dossier adminXXX/tabs/ de votre site.

 

Ensuite modifier les fichiers suivants : (dans les fichiers accompagnant ce module, les modifications apportées sont commentées par /*Mondial Relay*/)
